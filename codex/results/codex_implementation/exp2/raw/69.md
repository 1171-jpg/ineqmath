First, recall the AM-GM (Arithmetic Mean-Geometric Mean) inequality. For non-negative numbers \( x_1, x_2, \ldots, x_n \), we have:
\[
\frac{x_1 + x_2 + \ldots + x_n}{n} \geq \sqrt[n]{x_1 x_2 \cdots x_n}.
\]

### Step 2: Apply AM-GM to the cubic terms
Applying AM-GM to \( a^3 + b^3 + 2c^3 \):
\[
\frac{a^3 + b^3 + c^3 + c^3}{4} \geq \sqrt[4]{a^3 b^3 c^3 c^3} = \sqrt[4]{(abc)^3 c^3} = \sqrt[4]{c^6} = c^{\frac{3}{2}}.
\]
So, we can state that:
\[
a^3 + b^3 + 2c^3 \geq 4c^{\frac{3}{2}}.
