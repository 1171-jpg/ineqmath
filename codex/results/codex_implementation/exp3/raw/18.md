## Problem

Let $a, b, c, m, n$ be positive real numbers. Find the largest constant $C$ such that the following inequality holds for all $a, b, c, m, n \in \mathbb{R}^{+}$:
$$

$$

Applying the Cauchy-Schwarz inequality to the terms in the left-hand side:

\[

Let us denote \( S = \frac{a^2}{b(m a + n b)}+\frac{b^2}{c(m b + n c)}+\frac{c^2}{a(m c + n a)} \). From the Cauchy-Schwarz inequality, we have:

\[
S \cdot (b(m a + n b) + c(m b + n c) + a(m c + n a)) \geq (a + b + c)^2
