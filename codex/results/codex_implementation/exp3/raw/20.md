To find the largest constant \( C \) such that the following inequality holds for all \( a, b, c \geq 0 \) with \( ab + bc + ca > 0 \):

\[
\sqrt{\frac{a}{b^2 + bc + c^2}} + \sqrt{\frac{b}{c^2 + ca + a^2}} + \sqrt{\frac{c}{a^2 + ab + b^2}} \geq C \sqrt{\frac{ab + bc + ca}{(a+b)(b+c)(c+a)}}

### Step 1: Analyzing the Left Side

We denote 

\[
L = \sqrt{\frac{a}{b^2 + bc + c^2}} + \sqrt{\frac{b}{c^2 + ca + a^2}} + \sqrt{\frac{c}{a^2 + ab + b^2}}.

### Step 2: Analyzing the Right Side

We denote 

\[
R = C \sqrt{\frac{ab + bc + ca}{(a+b)(b+c)(c+a)}}

### Step 3: Exploring Cases

Given the symmetry in \( a, b, c \), we can consider specific cases to estimate \( C \).

#### Case 1: \( a = 1, b = 1, c = 0 \)

In this case,

\[

\[
