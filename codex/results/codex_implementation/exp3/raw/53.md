\[

### Step-by-Step Solution:

1. **Use AM-GM Inequality**: We can apply the Arithmetic Mean-Geometric Mean (AM-GM) inequality here.

   For any positive real numbers, AM-GM tells us that:
   \[
   \frac{x_1 + x_2 + ... + x_n}{n} \geq \sqrt[n]{x_1 x_2 ... x_n}
   \]
   with equality if and only if \( x_1 = x_2 = ... = x_n \).

2. **Apply AM-GM for Each Term**:
   For \( a^2 + b^2 + \sqrt{c} \):
   \[
   a^2 + b^2 + \sqrt{c} \geq 3 \sqrt[3]{a^2 \cdot b^2 \cdot \sqrt{c}} = 3 \sqrt[3]{a^2b^2c^{1/2}} = 3 \sqrt[3]{(ab)^2\sqrt{c}}.
   \]

   \[
   a^2 + b^2 + \sqrt{c} \geq 3 \sqrt[3]{(ab)^2 \cdot \frac{1}{\sqrt{ab}}} = 3 \sqrt[3]{ab}.

   \frac{ab}{a^2 + b^2 + \sqrt{c}} \leq \frac{ab}{3 \sqrt[3]{ab}} = \frac{1}{3} (ab)^{2/3}.
